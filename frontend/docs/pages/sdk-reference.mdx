# SDK Reference

The `invisible-wallet-sdk` is a TypeScript package providing a React hook and utility functions for integrating Veil into web applications.

**Package:** `invisible-wallet-sdk`  
**Source:** `sdk/src/`  
**Entry:** `dist/index.js` / `dist/index.d.ts`
